# libft
Basic C Library

## Getting Started

This is a rewrite of the most usefull functions in lib c

### Setup

Go on your project

`cd ./<your_project_path>`

Install de Library

`git clone https://github.com/acalleja2/libft.git <lib_name>`

Create the libft.a

`cd <lib_name>`
`make`

## Use

Compile your project with the Library

`gcc <your_project.c> <lib_name>`
